# Multi-Account Configuration Guide

This document provides instructions for setting up multiple accounts in the NodeSeek Signin application.

## Prerequisites

- Ensure you have the latest version of NodeSeek Signin installed.
- Have access to all accounts that you wish to configure.

## Steps to Configure Multiple Accounts

1. **Access the Account Settings**  
   Open the NodeSeek Signin application and navigate to the Account Settings section.

2. **Add New Accounts**  
   - Click the "Add Account" button.  
   - Enter the required credentials for each account (username, password, etc.).  
   - Click "Save" to add the account.

3. **Switch Between Accounts**  
   - After adding accounts, you can switch between them using the account dropdown in the top right corner of the application.  

4. **Manage Accounts**  
   - You can edit or remove accounts at any time through the Account Settings.  
   - Click on the account you wish to modify and make the necessary changes.

## Troubleshooting

- Make sure your internet connection is stable when adding or switching accounts.
- If you cannot log in, double-check your credentials.
- Refer to the FAQ section for common issues.

## Conclusion

You can now use the NodeSeek Signin application with multiple accounts seamlessly. If you encounter any problems, please refer to our support documentation or contact support.
